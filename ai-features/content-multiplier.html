Introduction

The Brightcove AI Content Multiplier can help you repurpose your long-form content to reach your audience with shorter videos.

Using the Content Multiplier

  1. Navigate to the media module and select your video.

  2. Click on the AI Button and select Clips or Crop to vertical.

    Content you can generate with this process:

    • Key Moments: Your video's most relevant moments.
    • Summaries: Key moments stiched toguether.
    • Reels: Key moments in vertical format.
    • Chapters: Longer video sections based on topic.

    Key moments are selected based on Relevant Topics, Audio intensity and visual queues to predict your most engaging snippets.

FAQ

What are the limitations of this feature?

  • Metadata is created only from text tracks; visual content is not analyzed.

  • If a video lacks captions, you can choose to generate them. If so, re-transcoding with current entitlements is required.

  • If a video has multiple language transcripts or audio tracks, the tool defaults to the English track or the most recently uploaded.

Is there a way to approve the content so it does not get automatically "pushed" to the public domain if it is tagged incorrectly?

  • Yes, all generated metadata requires your approval before it updates production values.

Can the Universal Translator be used to generate the text track for the metadata Optimizer?

  • Yes, Metadata optimizer can work with Universal Translator, Brightcove Autocaptions or text tracks generated by third party tools as long as they are in a standard captions format.

Will this tool work if I use captions generated from a tool outside Brightcove?

  • Yes, it supports captions generated by tools outside Brightcove.
